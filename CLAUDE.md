# Project Rules

## Writing Style
- Never use emdashes (—). Remove them if you ever see them. Use periods, commas, or rewrite the sentence instead.

## Deployment
- This is a GitHub Pages site hosted at: https://jeffcarp13.github.io/jeffscarpenter/
- All internal links must use relative paths (e.g., `liveearth.html` not `/liveearth.html`) because the site is served from a subdirectory.
